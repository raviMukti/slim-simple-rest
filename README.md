# REST API Demo
### Simple REST API

#### This Project Build With :
- Slim PHP 3.1
- Phinx
- DotEnv
- PHPUnit
- PostgreSQL

#### How To Run :
- `git clone this project`
- `composer install`
- Copy .env.example to .env
- Create DB and setup .env with your own-setup
- `vendor/bin/phinx migrate -e development`
- `composer start`
